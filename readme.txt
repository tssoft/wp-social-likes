== Description ==

Little, easy to use WordPress plugin that adds social networks sharing buttons to WordPress pages and posts.
Based on the Social Likes library by Artem Sapegin (http://sapegin.me/projects/social-likes).
Supported social networks include:
 * Facebook
 * Twitter
 * Google+
 * Pinterest
 * VK.com and other popular Russian social networks
Features:
 * Easy to install
 * Beautiful and all in one style
 * Won’t explode your page’s layout
 * Much faster than proprietary social buttons
 * Likes counters can be shown for each social network
 * Four different skins for social buttons
 * Buttons can be reordered easily
 * Horizontal or vertical buttons layout or single button mode with popup
 * Interactive preview of buttons look on the plugin settings page
Currently available in English and Russian. Any help with translating Social Likes to other languages will be greatly appreciated! (i18n with gettext)
